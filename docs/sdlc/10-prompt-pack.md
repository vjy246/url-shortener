# Prompt Pack

1. Requirements: "Act as a senior software architect. Do not write code. Return functional requirements, non-functional requirements, ambiguities, assumptions, risks, out of scope, and backlog."
2. Architecture: "Compare modular monolith, microservices, and serverless. Evaluate delivery risk, scalability, maintainability, testability, and operational complexity. Do not choose for me."
3. Decomposition: "Break the approved design into tasks with objective, dependencies, acceptance criteria, risks, validation, and AI suitability."
4. Tests: "Produce a test matrix covering happy paths, boundaries, negative cases, failures, concurrency, and regression."
5. Staff review: "Review architecture, transactions, DB correctness, error handling, performance, logging, security, testability, and unnecessary abstractions. Classify findings. Do not rewrite code."
6. Security: "Review URL validation, redirect abuse, SSRF boundaries, sensitive logging, secrets, auth, rate limiting, and deployment risks."
7. Assessor: "Score the repository against the rubric and identify weak evidence, unsupported claims, and decisions I cannot defend."
